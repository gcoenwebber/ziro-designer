# Symbols & Libraries

Symbols are the schematic face of a component. Each carries pins, fields
(reference, value, footprint), and a link to the footprint used on the board.

## The library chooser

The chooser presents a searchable tree of symbol libraries with a live preview.
Filter by name, keyword, or library to find the part you need.

## Fields

Every symbol has editable fields — at minimum a **Reference** (R1, U2 …) and a
**Value**. You can add custom fields (tolerance, MPN, datasheet) and template
field names project-wide via Schematic Setup.

## Multi-unit parts

Parts with several units (e.g. a quad op-amp) share a single reference across
their units; annotation keeps the numbering consistent.
