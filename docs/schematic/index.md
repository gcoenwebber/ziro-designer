# Schematic Capture — Overview

The **Schematic Editor** is where a design begins. You place symbols, wire them
into nets, and annotate the drawing into a complete, electrically-meaningful
circuit that the PCB Editor can turn into a board.

## What you can do

- Place symbols from the built-in libraries and wire pins into nets.
- Add labels, power symbols, no-connects, buses, and bus entries.
- Annotate references, run electrical checks, and manage multi-unit parts.
- Set up the sheet, title block, and formatting via **Schematic Setup**.

## Key concepts

- **Nets** — the electrical connections between pins, formed by wires, labels,
  and junctions.
- **Symbols** — the schematic representation of a component; each maps to a
  footprint for the board.
- **Annotation** — assigning reference designators (R1, C3, U2 …), shared
  across units of multi-unit parts.
