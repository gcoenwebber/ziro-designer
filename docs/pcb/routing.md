# Routing & Vias

## Routing tracks

Use the **Route Tracks** tool to draw copper pad-to-pad. The ratsnest line for
each connection disappears as you complete it. Track width follows the active
netclass unless you override it.

## Vias

Drop a **via** to move a track between layers. Via size and drill follow the
netclass defaults and can be overridden per net.

## Zones

Pour a **filled zone** (e.g. a ground plane) and it connects to matching-net
pads automatically, with proper clearances and thermal reliefs.

## Checking your work

Net highlighting, clearance outlines, and the selection heuristics help you
verify connectivity as you go.
