# Placing Footprints

## Moving and orienting

- `M` **move** a footprint; `G` **drag** it while keeping attached traces.
- `R` rotates; `F` flips it to the opposite side of the board.
- The **Properties** panel edits reference, value, lock state, position, and
  orientation numerically.

## Using the ratsnest

The ratsnest shows which pads still need connecting. It updates live as you
move parts, so you can arrange components to minimize crossing connections
before you route.

## Locking and grouping

Lock a footprint to prevent accidental moves, or group items so they move
together.
