# PCB Layout — Overview

The **PCB Editor** turns your schematic into a physical board. Footprints arrive
linked to the schematic by the **ratsnest**; you place them, route the copper,
pour zones, and generate manufacturing outputs.

## What you can do

- Place and orient footprints, edit their reference/value/position.
- Route tracks and vias across layers with live ratsnest feedback.
- Draw graphics, add text, pour filled zones, and measure.
- Manage the layer stack and appearance; run the board through checks.
- Set board-wide rules via **Board Setup**.

## Key concepts

- **Ratsnest** — the thin lines showing required connections not yet routed.
- **Nets & netclasses** — electrical groups with their own width/clearance
  rules.
- **Layers** — copper, silkscreen, mask, and more, toggled in the Appearance
  panel.
