# The Interface

Ziro Designer opens to a **home dashboard** — your project manager — and each
editor opens as a full-screen frame from there. The chrome is deliberately
familiar to anyone who has used a desktop EDA tool.

## The workspace at a glance

- **Home / Project Manager** — create, open, and organize projects. Your files
  live in the cloud and sync automatically.
- **Editor frames** — Schematic, PCB, Gerber, Calculator, and more. Each opens
  in its own frame with its own toolbars.
- **Top toolbar** — file actions, undo/redo, and editor-wide commands.
- **Left toolbar** — the drawing and editing tools for the active editor.
- **Right docks** — context panels: properties, layers/appearance, selection
  filters, and libraries.
- **Status bar** — cursor coordinates, grid, active tool, and units.

## Common conventions

- **Hotkeys** mirror the desktop workflow — `M` to move, `G` to drag, `R` to
  rotate, `Esc` to cancel, and so on.
- **Undo/redo** is available everywhere with `Ctrl+Z` / `Ctrl+Y`.
- **Autosave** persists your work to the project file manager continuously; use
  _Save a Copy_ to download a local file.

::: tip
Every editor shares the same shell, so once you learn the layout in one, the
others feel immediately familiar.
:::
