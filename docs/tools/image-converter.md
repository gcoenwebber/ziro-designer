# Image Converter

The **Image Converter** turns bitmap artwork — logos, labels, decorative
graphics — into board features you can place on silkscreen, copper, or other
layers.

## Workflow

1. Import a bitmap (PNG, JPG, …).
2. Adjust threshold and conversion settings to get a clean black-and-white
   result.
3. Export as a footprint or graphic and place it on the board.
