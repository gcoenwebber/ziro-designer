# Project-manager icons

KiCad's own icons (dark-theme variants from `resources/bitmaps_png/sources/dark/`
in the [KiCad source tree](https://gitlab.com/kicad/code/kicad)), vendored
unmodified for the project manager's left toolbar and file tree:

- toolbar: `new_project_from_template`, `open_project`, `zip`, `unzip`,
  `refresh`, `directory_browser`
- file tree: `project_kicad` (`.kicad_pro`), `icon_eeschema_24` (`.kicad_sch`),
  `library` (`.kicad_sym`), `directory`
