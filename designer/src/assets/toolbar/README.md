# Toolbar icons

KiCad's own toolbar icons (dark-theme variants from
`resources/bitmaps_png/sources/dark/` in the
[KiCad source tree](https://gitlab.com/kicad/code/kicad)), vendored unmodified.
Each schematic-editor toolbar button maps to the bitmap named in KiCad's
`SCH_ACTIONS` / `ACTIONS` `.Icon(BITMAPS::…)` definitions (see
`src/ui/toolbarIcons.ts` for the id → bitmap map).
