# Launcher icons

These are KiCad's own application icons (the dark-theme variants from
`resources/bitmaps_png/sources/dark/` in the
[KiCad source tree](https://gitlab.com/kicad/code/kicad)), vendored unmodified
for the project-manager launcher tiles.

They are part of KiCad and licensed under **GPL-3.0-or-later**, the same license
